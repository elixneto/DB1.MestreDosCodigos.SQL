# 7. (Oracle, SQL Server, Mysql, PostgreSQL) Dado que você tenha a tabela "User" em seu banco de dados, e que tem os seguintes dados cadastrados:

idUser    codUser

Opção 5: "SELECT * FROM user WHERE codUser LIKE '10%'";
